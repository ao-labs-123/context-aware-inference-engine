# ステージ3 ― 因果関係の推論（Causal Direction Analysis）
## 概要：
本ステップでは、ステージ1で特定した主語（エージェント）を基盤とし、対話の論理的流れをマッピングします。接続詞、接続副詞、および接続助詞（「～から」「～ので」「～ため」など）を解析することで、事象間の因果関係の向きを正確に特定し、前提と結論の取り違えを防ぎます。

## 主要ポイント：
**1. 論理マーカーのマッピング**：
文中の「接続マーカー」を識別し、文構造を「原因と結果」「論理の反転（逆接）」「事象の順序」などのペアに分類します。

**2. 方向性の依存関係**：
原因と結果をステージ1で特定したエージェントに紐付けることで、「誰が」「なぜ」行動したのかという動機を明確にし、因果関係が常に正しい主体に帰属するようにします。

**3. 構造の曖昧性解消**：
複数の事象が関連する複雑な文において、結果と背後にある動機を混同しないように論理的な分離を行います。
