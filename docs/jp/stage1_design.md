# ステージ1 ― 対話AIにおける主語の推定

## 概要：
本ステップでは、主語が省略された文を解釈するモデルの能力を最適化することに焦点を当てます。
心理動詞（例：「思う」「感じる」「気づく」など）が、一貫して話者を主たる主体（エージェント）としてマッピングする言語パターンを体系化することで、モデルは主語の特定における曖昧さを排除し、言語を問わず対話の精度を飛躍的に向上させます。

## 主要ポイント：
**1. 心理動詞のデフォルト設定**：
主語が明示されていない文で心理動詞（例：「思う」「感じる」「〜したい」「期待する」）が現れた場合、デフォルトで話者をその主語（エージェント）として割り当てます。

**2. 証拠性および帰属のオーバーライド（優先設定）**：
文に証拠性を示す表現や間接話法（例：「〜らしい」「〜と言われている」「〜と聞いた」「〜だそうだ」）が含まれる場合、話者のデフォルト設定を上書きし、その主体を第三者として扱います。

**3. 曖昧さの低減**：
この決定論的なマッピングは、非公式でコンテキスト（文脈）に依存した対話における主語の曖昧さを効果的に解消し、多段階の論理推論を行うための基礎となります。

## 論理比較：省略主語 vs. 証拠性によるオーバーライド
| 入力文 | 論理プロセス | 結果 |
|--|--|--|
| 「変だと思った。」 | 心理動詞 + ゼロ主語 → [デフォルト：話者] | AIは「私」を主語として割り当てる。 |
| 「どうやら変だと思ったらしい。」 | 心理動詞 + ゼロ主語 + 証拠性の表現 → [オーバーライド：第三者] | AIは「彼/彼女/彼ら」を主語として割り当てる |
